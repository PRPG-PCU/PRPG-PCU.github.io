
# 1주차 개발일지

1. 모델, 이미지, 사운드 리소스 탐색 20% 

2. 시작화면 구현 80% -> 배경이미지에 특수효과를 구현

4. 화면간 이동 구현 100%

5. 스토리 1차 구현 20% -> 전체틀 구현

6. 캐릭터 이동 구현 50% -> 충돌 구현
